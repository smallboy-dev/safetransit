# SafeTransit AI

SafeTransit AI is a telecom-powered urban mobility platform designed for African cities, submitted for the GSMA Africa Ignite Hackathon 2026.

## Theme: Mobility, Transport & Urban Services

The platform integrates Nokia's Network-as-Code APIs to enhance passenger safety, trust, and operational efficiency in urban transport.

## Nokia Network-as-Code API Usage

We leverage the following Nokia APIs to ensure a secure and reliable experience:

- **SIM Swap API**: Detects fraudulent SIM changes to protect driver accounts.
- **Device Swap API**: Identifies suspicious device changes to prevent unauthorized access.
- **Number Verification API**: Authenticates driver phone numbers during registration and login.
- **Location Retrieval API**: Provides real-time vehicle tracking for passengers and operators.
- **Geofencing API**: Automates alerts for arrivals, departures, and route deviations.
- **Location Verification API**: Confirms device location accuracy to prevent GPS spoofing.
- **Device Status / Reachability API**: Monitors driver connectivity to ensure reliable tracking.

## Expected Benefits

- **Passenger Safety**: Verified drivers and fraud detection significantly reduce risks.
- **Increased Trust**: Reliable tracking and real-time alerts improve confidence in public transport.
- **Operational Efficiency**: Real-time monitoring helps reduce waiting times and optimize routes.
- **Digital Inclusion**: Enables smarter, safer mobility for underserved communities in Africa.

## Technology Stack

- **Frontend**: Flutter (Mobile)
- **Backend**: Firebase (Auth, Firestore, Messaging)
- **Integration**: Nokia Network-as-Code APIs (Network programmability)
